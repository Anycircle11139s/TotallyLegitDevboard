# TotallyLegitDevboard

TLD (Totally legit fevboard) Overview

TLD is a custom RP2040-based devboard that can be used for lots of cool hardware and software projects. It follows the same pinout as the Raspberry Pi Pico and is fully feature-compatible. It is based on the theme of it beinng low quality and barely working and I have some funny joes on the PCB.

It will also include headers soldered on.

Features
16MB onboard flash memory
Convenient reset button for easy flashing
Custom jokes on both sides of the PCB
